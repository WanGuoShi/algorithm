## 简介

这个库主要是用来记录一些常见的算法和数据结构方面的问题，比如：常见的排序算法（冒泡排序、
选择排序、插入排序、归并排序、堆排序、快速排序、希尔排序、桶排序等，还有一些面试中常见的
算法题目，比如：选择字符串问题、字符串转整形、回文判断等。

## 目标

记录自己学习算法的过程，同时共享自己的学习心得，与广大网友交流共同进步。

## 目前状态

目前这个库正在总结常见的排序算法，已经完成冒泡排序、选择排序、插入排序和归并排序的简单
实现。
